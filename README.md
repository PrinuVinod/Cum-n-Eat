# Restaurant-App
This is a menu, ordering, etc Web App for Restaurants.

## Installation
```
npm install express mongoose body-parser dotenv ejs
```

## Paths
1. Home Screen
```
/
```
3. To check out the Menu
```
/menu
```
3. To get Ordered List
```
/menu/get-orders
```
4. To add new Item to the Menu
```
/additem
```
